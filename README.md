# Random Background Color Generator

A simple web app that generates a random background color when a button is clicked and displays the color code on screen.

## How to Use

1. Open the `index.html` file in a web browser.
2. Click the "Generate Random Color" button to change the background color.
3. The color code will be displayed on the screen.

## Files

- `index.html`: The main HTML file for the web app.
- `style.css`: The CSS file for styling the web app.
- `script.js`: The JavaScript file for the functionality.

## Dependencies

This project does not require any external dependencies. It only uses HTML, CSS, and JavaScript.
